## Workflow Recomendado para Link Building

### 1. Análisis de Competencia
```
Ubersuggest → Ver backlinks de competidores (ej: localiza.com, hertz.com.co)
SEMRush → Backlink gap analysis
Majestic → Trust Flow de dominios enlazantes
```

### 2. Búsqueda de Oportunidades
```
Buzzsumo → Contenido viral en nicho para replicar
Storybase → Preguntas que hace la gente
Spamzilla → Dominios expirados con backlinks relevantes
```

### 3. Creación de Contenido Link-worthy
```
ChatGPT/Growthbar → Generar contenido
Canva → Infografías
Envato Elements → Assets visuales
```

### 4. Outreach y Seguimiento
```
[Manual] → Contactar sitios identificados
Nightwatch → Trackear rankings
```

### 5. Indexación de Nuevos Backlinks
```
One Hour Indexing → Para backlinks importantes
Linklicious/Indexification → Para volumen
```

## Notas Importantes

1. **Credenciales compartidas**: Se actualizan cada 5-7 días
2. **Fechas de actualización**: Las fechas tipo "27/12" o "05/03" indican última actualización de credenciales
3. **Pack Full**: Algunas herramientas requieren membresía "Full"
4. **Acceso vía Telegram**: Algunas herramientas requieren solicitar acceso por Telegram
5. **Métodos especiales**: Varias herramientas usan login por enlace especial, no usuario/contraseña
